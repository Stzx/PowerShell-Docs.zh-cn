### <a name="scope-issues"></a><span data-ttu-id="7aba8-266">作用域问题</span><span class="sxs-lookup"><span data-stu-id="7aba8-266">Scope issues</span></span>

<span data-ttu-id="7aba8-267">这也有助于缓解作用域问题。</span><span class="sxs-lookup"><span data-stu-id="7aba8-267">This also helps mitigate scoping issues.</span></span> <span data-ttu-id="7aba8-268">在本例中，我们在循环中反复将值赋予 `$result`。</span><span class="sxs-lookup"><span data-stu-id="7aba8-268">In that example, we assign values to `$result` over and over in a loop.</span></span> <span data-ttu-id="7aba8-269">但由于 PowerShell 允许来自函数外部的变量值渗透到当前函数的作用域中，因此在函数内部对其初始化可减少可能通过这种方式引入的 bug。</span><span class="sxs-lookup"><span data-stu-id="7aba8-269">But because PowerShell allows variable values from outside the function to bleed into the scope of the current function, initializing them inside your function mitigates bugs that can be introduced that way.</span></span>

<span data-ttu-id="7aba8-270">如果函数中未初始化的变量设置为父作用域中的一个值，则将不会是 `$null`。</span><span class="sxs-lookup"><span data-stu-id="7aba8-270">An uninitialized variable in your function is not `$null` if it's set to a value in a parent scope.</span></span>
<span data-ttu-id="7aba8-271">父作用域可能是调用你的函数并使用相同变量名称的另一个函数。</span><span class="sxs-lookup"><span data-stu-id="7aba8-271">The parent scope could be another function that calls your function and uses the same variable names.</span></span>

<span data-ttu-id="7aba8-272">如果我使用同一个 `Do-something` 示例并删除循环，最终就会看到类似于以下示例的内容：</span><span class="sxs-lookup"><span data-stu-id="7aba8-272">If I take that same `Do-something` example and remove the loop, I would end up with something that looks like this example:</span></span>

```powershell

    $result = 'ParentScope'

function Do-Something

    {
        $result = Get-Something -ID $node
```

<span data-ttu-id="7aba8-273">如果对 `Get-Something` 的调用引发异常，那么我的 `$null` 检查将从 `Invoke-Something` 找到 `$result`。</span><span class="sxs-lookup"><span data-stu-id="7aba8-273">If the call to `Get-Something` throws an exception, then my `$null` check finds the `$result` from `Invoke-Something`.</span></span> <span data-ttu-id="7aba8-274">初始化函数内的值会缓解此问题。</span><span class="sxs-lookup"><span data-stu-id="7aba8-274">Initializing the value inside your function mitigates this issue.</span></span>

<span data-ttu-id="7aba8-275">命名变量很难，因此，作者通常会在多个函数中使用相同的变量名称。</span><span class="sxs-lookup"><span data-stu-id="7aba8-275">Naming variables is hard and it's common for an author to use the same variable names in multiple functions.</span></span> <span data-ttu-id="7aba8-276">我一直在使用 `$node`、`$result` 和 `$data`。</span><span class="sxs-lookup"><span data-stu-id="7aba8-276">I know I use `$node`,`$result`,`$data` all the time.</span></span> <span data-ttu-id="7aba8-277">因此，来自不同作用域的值很容易出现在它们不该出现的地方。</span><span class="sxs-lookup"><span data-stu-id="7aba8-277">So it would be very easy for values from different scopes to show up in places where they should not be.</span></span>

## <a name="redirect-output-to-null"></a><span data-ttu-id="7aba8-278">将输出重定向到 $null</span><span class="sxs-lookup"><span data-stu-id="7aba8-278">Redirect output to $null</span></span>

### <a name="out-null"></a><span data-ttu-id="7aba8-282">Out-Null</span><span class="sxs-lookup"><span data-stu-id="7aba8-282">Out-Null</span></span>

<span data-ttu-id="7aba8-283">Out-Null 命令是将管道数据重定向到 `$null` 的内置方法。</span><span class="sxs-lookup"><span data-stu-id="7aba8-283">The Out-Null command is the built-in way to redirect pipeline data to `$null`.</span></span>

### <a name="assign-to-null"></a><span data-ttu-id="7aba8-284">分配给 $null</span><span class="sxs-lookup"><span data-stu-id="7aba8-284">Assign to $null</span></span>

<span data-ttu-id="7aba8-285">可以将命令的结果分配给 `$null`，以获得与使用 `Out-Null` 相同的结果。</span><span class="sxs-lookup"><span data-stu-id="7aba8-285">You can assign the results of a command to `$null` for the same effect as using `Out-Null`.</span></span>

<span data-ttu-id="7aba8-286">由于 `$null` 是常量值，因此永远不能覆盖它。</span><span class="sxs-lookup"><span data-stu-id="7aba8-286">Because `$null` is a constant value, you can never overwrite it.</span></span> <span data-ttu-id="7aba8-287">我不喜欢它在代码中的样子，但它的执行速度通常比 `Out-Null` 快。</span><span class="sxs-lookup"><span data-stu-id="7aba8-287">I don't like the way it looks in my code but it often performs faster than `Out-Null`.</span></span>

### <a name="redirect-to-null"></a><span data-ttu-id="7aba8-288">重定向到 $null</span><span class="sxs-lookup"><span data-stu-id="7aba8-288">Redirect to $null</span></span>

<span data-ttu-id="7aba8-289">你还可以使用重定向运算符将输出发送到 `$null`。</span><span class="sxs-lookup"><span data-stu-id="7aba8-289">You can also use the redirection operator to send output to `$null`.</span></span>
