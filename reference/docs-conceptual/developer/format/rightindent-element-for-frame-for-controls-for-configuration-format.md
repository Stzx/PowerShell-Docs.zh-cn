# <a name="rightindent-element-for-frame-for-controls-for-configuration-format"></a>RightIndent Element for Frame for Controls for Configuration (Format)

指定数据从右边缘向外移动的字符数。 此元素在定义可供格式设置文件中的所有视图使用的公共控件时使用。

